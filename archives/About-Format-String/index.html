Format String 花式玩法

1. 解決PIE

Stack Memory Layout

Stack的分布大致上是:

==========================

 Enviorment Varibles

rsp===>(data)
(data)
(data)
Stack Canary
rbp===>(data)
Saved RBP
return address

==========================

Solution

因此可以構造 %(order)$p 的Payload去 Leak Return Address

然後將__libc_start_main的offset減掉之後，做後續的ROP

p.s. RBP之下的東西會先經過shared library再到Heap

3. 搭配Heap組合技

小知識

在需要輸出的字串過大時(>100000), 會用Heap來存放要輸出的東西

所以可以用%(count)c構造payload: %100000c產生長度夠的字串

然後把__free_hook或是__malloc_hook的GOT蓋掉, 這樣程式在malloc()/free()時就會去執行我們蓋掉的值
