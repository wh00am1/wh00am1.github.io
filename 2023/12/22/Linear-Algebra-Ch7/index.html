Hermitian, Unitary, Normal matrices

  • Conjugate transpose : A∗=ATˉA^{*} = \bar{A^T}A∗=ATˉ
  • Unitary matrix : AA∗=A∗A=IAA^* = A^*A = IAA∗=A∗A=I or it’s conjugate equals to inverse
    • Has orthogonal row vectors
  • Hermitian matrix : A∗=AA^*=AA∗=A : diagonal entries must be real
    • Has orthogonal eigenvectors
    • Unitarily diagonalize : Gram-Schmidt the eigenspace, then find P to diagonalize
