Evaluate by Cofactor Expansion

Expand the first row for example

det(M)=m11C11+m12C12+...+m1nC1ndet(M) = m_{11}C_{11}+m_{12}C_{12}+...+m_{1n}C_{1n} det(M)=m11​C11​+m12​C12​+...+m1n​C1n​

Evauate 3x3 matrix quick method

det(M)=m11m22m33+m12m23m31+m13m21m33−m13m22m31−m12m21m33−m11m23m32det(M) = m_{11}m_{22}m_{33}+m_{12}m_{23}m_{31}+m_{13}m_{21}m_{33}-m_{13}m_{22}m_{31}-m_{12}m_{21}m_{33}-m_{11}m_{23}m_{32} det(M)=m11​m22​m33​+m12​m23​m31​+m13​m21​m33​−m13​m22​m31​−m12​m21​m33​−m11​m23​m32​

Evaluate by Row Reduction

Determinant of Elementary Matrix

  • Let EEE be n×nn \times nn×n Elementary Matrix
  • If:
    1. Multiply a row by kkk, then det(E)=kdet(E)=kdet(E)=k
    2. Interchanging 2 rows, then det(E)=−1det(E)=-1det(E)=−1
    3. Adding a row(k times), then det(E)=1det(E)=1det(E)=1

Evaluation

  1. Reduce to row echelon
  2. Evaluate by :

det(M)=tr(M)det(M) = tr(M) det(M)=tr(M)

Properties

With n×nn \times nn×n matrix A, B :

  1. det(kA)=kndet(A)det(kA)=k^n det(A)det(kA)=kndet(A)
  2. det(A)det(B)=det(AB)det(A)det(B)=det(AB)det(A)det(B)=det(AB)
  3. seperate entries of a row/column: det(A1)+det(A2)det(A_1)+det(A_2)det(A1​)+det(A2​)

Cramer’s Rule

Basic

With n×nn \times nn×n matrix A, 1×n1 \times n1×n matrix B and Ax=BAx=BAx=B linear system:

  1. The solution is:

xn=det(An)det(A)x_n = \frac{det(A_n)}{det(A)} xn​=det(A)det(An​)​

Properties

  1. Can use the method only when A is invertible.
